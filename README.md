## Prerequisites

- [Node LTS](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/pt-BR/)

## Installing

```bash

  # Clone repository
  $ git clone https://github.com/pereiradilson/carousel.git

  # Enter folder
  $ cd carousel

  # Install dependences
  $ yarn

  #Execute
  $ yarn start

```
